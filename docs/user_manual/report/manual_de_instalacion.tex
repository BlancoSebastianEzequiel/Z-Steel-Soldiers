Para la correcta instalación de la aplicación, se deben seguir los siguientes
pasos:
\begin{itemize}
    \item Descargar la librerías \code{SDL} y \code{SDL image}, mediante los
    siguientes comandos:
    \begin{itemize}
        \item \code{sudo apt-get install libsdl1.2-dev}
        \item \code{sudo apt-get install libsdl-image1.2-dev}
    \end{itemize}
    \item Abrir una terminal en el \code{root} del repositorio ejecutar el
    servidor con el siguiente comando: \code{sh scripts/run\_server.sh}
    \item Abrir otra una terminal en el \code{root} del repositorio ejecutar un
    cliente con el siguiente comando: \code{sh scripts/run\_client.sh}
\end{itemize}
