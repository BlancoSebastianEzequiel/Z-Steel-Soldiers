Para el correcto funcionamiento de la aplicación se deben tener los siguientes
programas instalados:
\begin{itemize}
    \item \underline{\textbf{Sistema operativo:}} Ubuntu 16.04
    \item Compilador de ISO C++11
    \item Instalación de las librerías \code{SDL 1.2} y \code{SDL-image 1.2},
    de no tenerlas se requiere escribir los siguientes comandos en la terminal:
    \begin{itemize}
        \item \code{sudo apt-get install libsdl1.2-dev}
        \item \code{sudo apt-get install libsdl-image1.2-dev}
    \end{itemize}
    \item Para depurar el programa debe utilizarse \code{GNU gdb 7.11.1}
\end{itemize}
